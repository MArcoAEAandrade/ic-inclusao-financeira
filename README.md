# Inclusão Financeira de Jovens Periféricos

Este repositório contém os notebooks e análises da Iniciação Científica
**"Inclusão Financeira de Jovens Periféricos: Comportamento e Barreiras"**.

## Objetivo
Explorar dados secundários oficiais (PNADC, Relatórios Financeiros)
para contextualizar barreiras de acesso ao sistema financeiro,
com foco em juventude periférica.

## Estrutura
- `notebooks/` – análises exploratórias e cruzamentos
- `data/` – dados brutos e tratados (quando aplicável)
- `figures/` – gráficos utilizados na pesquisa
